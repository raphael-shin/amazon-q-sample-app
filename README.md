# Amazon Q Sample App - Todo API

FastAPI 기반의 Todo 관리 API 백엔드 서버입니다.

## 기능

- Todo 항목 생성, 조회, 수정, 삭제 (CRUD)
- Swagger UI를 통한 API 테스트
- Docker 컨테이너 지원

## 시작하기

### 환경 설정

```bash
# 환경 설정 스크립트 실행
./setup_env.sh
```

### 서버 실행

```bash
# 서버 실행 스크립트
./run.sh
```

### API 문서

서버 실행 후 다음 URL에서 Swagger UI를 통해 API를 테스트할 수 있습니다:
- http://localhost:8000/docs
